affentanz

cross-platform framework for interactive 3d-applications

Download this project as a .zip file Download this project as a tar.gz file

about.

affentanz engine is an open-source framework for building interactive 3d applications like games for different platforms. It's work in progress and not all parts are finished yet. You're invited to contribute on github!

features.

  • 3D-Rendering
  • Physics & Collision
  • Input
  • 2D-GUI
  • Entity-Component System
  • Event System
  • Lua Scripting
  • Cross-Platform: Windows, Linux, Mac, Android, iOS

download and build.

Have a look at the wiki at https://github.com/eikelangbehn/affentanz/wiki and the build instructions https://github.com/eikelangbehn/affentanz/wiki/Build-Engine
